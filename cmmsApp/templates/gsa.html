IEC 61131 programming standards

International Standard IEC 61131 defines basic software architecture and programming languages used in PLC control programs:

  • Ladder Diagram (LD)
  • Function Block Diagram (FBD)
  • Structured Text (ST)
  • Instruction List (IL)
  • Sequential Function Chart (SFC)
